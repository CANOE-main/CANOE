# Related Repositories

The CANOE project consists of several interconnected repositories:

- **[temoa](https://github.com/CANOE-main/temoa.git)**: Tools for Energy Model Optimization and Analysis. The core modeling framework.
- **[canoe_interface](https://github.com/CANOE-main/canoe_interface)**: Interface to filter national model data
- **[Onboarding Materials](https://github.com/CANOE-main/Onboarding)**: Onboarding materials for the CANOE project
- **[CANOE Swap](https://github.com/CANOE-main/CANOE_SWAP)**: Utility script that allows users to swap data.
- **[representative_periods](https://github.com/CANOE-main/representative_periods.git)**: Workflow for selecting representative days for the model.
- **[canoe-transportation](https://github.com/CANOE-main/canoe-transportation.git)**: Transportation sector data and processing scripts.
- **[canoe-electricity](https://github.com/CANOE-main/canoe-electricity.git)**: Electricity sector data processing and input generation.
- **[canoe-industry](https://github.com/CANOE-main/canoe-industry.git)**: Industry sector data processing and input generation.
- **[canoe-residential](https://github.com/CANOE-main/canoe-residential.git)**: Residential sector data processing and input generation.
- **[canoe-commercial](https://github.com/CANOE-main/canoe-commercial.git)**: Commercial sector data processing and input generation.
- **[canoe-fuels](https://github.com/CANOE-main/canoe-fuels.git)**: Fuels sector data processing and input generation.
- **[canoe-agriculture](https://github.com/CANOE-main/canoe-agriculture.git)**: Agriculture sector data processing and input generation.
- **[canoe-cef](https://github.com/CANOE-main/canoe-cef.git)**: CEF (Canada's Energy Futures) sector data and constraints structure.
