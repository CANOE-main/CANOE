# Contributing

We are happy that you're interested in contributing to CANOE!

## Resources
* [Model Documentation](index.md)
* [Issue Tracker](https://groups.google.com/forum/#!forum/temoa-projects)

## Submitting Changes
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Commit your changes with descriptive messages.
4. Submit a Pull Request.

## Code Style
Please ensure your code follows the existing style conventions of the project.
